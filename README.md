Mediconnect - Plateforme de Téléconsultation

​Mediconnect est un projet d'application web permettant de mettre en relation les patients et les médecins. Le but est de proposer une interface simple et efficace pour gérer les consultations à distance.

​État actuel du projet
​Le projet contient une page d'accueil en PHP ainsi qu'un système d'inscription et de connexion. Les interfaces sont différenciées selon que l'utilisateur est un patient ou un médecin. Le design a été travaillé pour être lisible aussi bien sur ordinateur que sur téléphone portable.

​Structure des fichiers

​Image : Dossier contenant les logos et les visuels.

​Page_d_Acceuil.php : Page principale du site pour l'interface patient

​Connexion.html : Page de connexion et de création de compte.

​CSS : Dossier regroupant l'ensemble des styles du site.

​Dashboard.php : Espace dédié aux médecins.
